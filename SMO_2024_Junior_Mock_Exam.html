Section A — Multiple Choice Questions (Q1–5)
Question 1
If integers \(a, b, c\) satisfy \((a-b)^{10} + (a-c)^{10} = 1\), find the value of \(|a-b|+|b-c|+|c-a|\).
Question 2
Given that real numbers \(a, b, c\) satisfy \(a+b+c=3\) and \(a^2+b^2+c^2=4\), find the value of \[\dfrac{a^2+b^2}{2-c}+\dfrac{b^2+c^2}{2-a}+\dfrac{c^2+a^2}{2-b}.\]
Question 3
\(y = x^2+bx+c\) passes through \(A(m, n)\) and \(B(m-8, n)\). If it only has one common point with the \(x\)-axis, find \(n\).
Question 4
In rectangle \(ABCD\), \(AD=5\), \(AB=10\). Points \(E\) and \(F\) are outside the rectangle. If \(BE=DF=4\) and \(AF=CE=3\), find \(EF\).
Question 5
Calculate: \[\dfrac{1}{3+1}+\dfrac{2}{3^2+1}+\dfrac{4}{3^4+1}+\dfrac{8}{3^8+1}+\cdots+\dfrac{2^{2006}}{3^{2^{2006}}+1}.\]
Section B — Short Answer Questions (Q6–25)
Question 6
Let \(\lfloor x \rfloor\) denote the largest integer not greater than \(x\), and \(x - \lfloor x \rfloor\) gives the decimal (fractional) part of \(x\). Given \(t = \dfrac{1}{2-\sqrt{3}}\), let \(a\) and \(b\) be the decimal parts of \(t\) and \(-t\) respectively. Find \(\dfrac{1}{2b} - \dfrac{1}{a}\).
Enter as a fraction: e.g. 1/2
Question 7
How many different 4-digit numbers \(\overline{abcd}\) with \(a+c = b+d\) can be formed using the digits 1, 2, 3, and 4, where repeated digits are allowed?
Question 8
Suppose \(x - y = 1\). Find the value of \[x^4 - xy^3 - x^3y - 3x^2y + 3xy^2 + y^4.\]
Question 9
Simplify: \[\dfrac{\left(1-\dfrac{1}{2^2}\right)\!\left(1-\dfrac{1}{3^2}\right)\!\left(1-\dfrac{1}{4^2}\right)\cdots\!\left(1-\dfrac{1}{2005^2}\right)}{\left(1-\dfrac{1}{2}\right)\!\left(1-\dfrac{1}{3}\right)\!\left(1-\dfrac{1}{4}\right)\cdots\!\left(1-\dfrac{1}{2005}\right)}.\]
Question 10
Find the value (in simplest form) of \(\sqrt{9+4\sqrt{5}}-\sqrt{9-4\sqrt{5}}\).
Question 11
If \(x > 0\) and \(\left(x + \dfrac{1}{x}\right)^2 = 25\), find the value of \(x^3 + \dfrac{1}{x^3}\).
Question 12
In the diagram, \(AB \parallel EF \parallel DC\). Given that \(AC + BD = 250\), \(BC = 100\), and \(EC + ED = 150\), find \(CF\).
Question 13
Consider the following "star" figure. Given that \(\angle p+\angle q+\angle r+\angle s+\angle t=500°\) and \(\angle A+\angle B+\angle C+\angle D+\angle E=x°\), find the value of \(x\).
Question 14
What is the product of the real roots of the equation \[\dfrac{x^2+90x+2027}{3} = \sqrt{x^2+90x+2055}\,?\]
Question 15
What is the minimum value of \(\sqrt{x^2+4} + \sqrt{(12-x)^2+9}\,?\)
Write as sqrt(145) or √145
Question 16
Side length of square \(ABCD\) is 1. Extend side \(BC\) to point \(E\) such that \(CE = 1\). Connect \(A\) and \(E\); let \(F\) be the intersection of \(AE\) with \(CD\). Connect \(B\) and \(F\); connect \(D\) and \(E\). Extend \(BF\) to point \(G\) on \(DE\). Find the length of \(BG\).
Write as 2sqrt(5)/3 or 2√5/3
Question 17
A "peaceful number" can be represented by the difference of cubes of two consecutive odd numbers. For example, 2 and 26 are peaceful numbers because \(2 = 1^3 - (-1)^3\) and \(26 = 3^3 - 1^3\). How many peaceful numbers that are also positive integers are there less than 2016?
Question 18
Positive integers \(a\) and \(b\) satisfy \(a = b - 2005\). If \(x^2 - ax + b = 0\) has positive integer solutions, find the minimum value of \(a\).
Question 19
In parallelogram \(ABCD\), \(\angle ABC = 72°\). \(AF \perp BC\) at point \(F\), and \(AF\) intersects diagonal \(BD\) at point \(E\). If \(DE = 2AB\), find \(\angle AED\) in degrees.
Question 20
Find the number of integer solution sets \((x, y)\) of \(x^2 + 2xy + 3y^2 = 34\).
Question 21
If the roots of \(x^2 - 3x - 1 = 0\) are also roots of \(x^4 + ax^2 + bx + c = 0\), find the value of \(a + b - 2c\).
Question 22
Positive numbers \(a\) and \(b\) satisfy \(\dfrac{a-1}{b} + \dfrac{b-1}{a} + 1 = 0\). Find the value range of \(a+b\).
Write an inequality, e.g. 1 < a+b ≤ 4/3  (this question is checked manually)
Question 23
\(O\) is the intersection of diagonals \(AC\) and \(BD\) of quadrilateral \(ABCD\). Given \(\angle BAD + \angle ACB = 180°\), \(BC = 3\), \(AD = 4\), \(AC = 5\), \(AB = 6\), find \(\dfrac{DO}{OB}\).
Write as a fraction: 10/9
Question 24
How many possible values are there for \(m\) (non-negative integer) such that \(5 \times 2^m + 1\) is a perfect square number?
Question 25
